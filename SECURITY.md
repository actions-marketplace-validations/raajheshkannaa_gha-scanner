# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in this scanner, please report it responsibly.

**Do not open a public issue for security vulnerabilities.**

Instead, use [GitHub's private vulnerability reporting](https://github.com/raajheshkannaa/gha-scanner/security/advisories/new) to submit your report.

You can expect an initial response within 48 hours.

## Scope

Security reports are accepted for:
- The scanner's web application and API
- The CLI tool
- The scanning engine and check logic
- Dependencies with exploitable vulnerabilities

## Out of Scope

- False positives or severity disagreements in scan results (use the [false positive issue template](https://github.com/raajheshkannaa/gha-scanner/issues/new?template=false-positive.md) instead)
- Vulnerabilities in scanned repositories (those are the repositories' responsibility)
- Rate limiting or availability issues on the hosted instance
